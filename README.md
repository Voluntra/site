<h4 align="center">Connecting students to volunteer organizations</h4>

## Running locally

```bash
# Clone this repository
$ git clone https://github.com/Voluntra/voluntra.git

# Go into the repository
$ cd voluntra

# Install dependencies
$ yarn

# Run the app
$ yarn dev
```
